# Linmas 0.1.9

## Evaluation foundation

- Added versioned synthetic evaluation cases and normalized `ReviewResult` validation.
- Added deterministic contract, finding, evidence, remediation, verification, check, and safety scoring.
- Added an offline evaluation command suitable for CI without network access or credentials.
- Added a bounded advisory Claude evaluation runner and trusted scheduled workflow.

Passing included cases does not prove that a response or project is secure. Live evaluation is advisory and never approves changes or releases. Human review remains required.
