# Linmas 0.2.0

## `linmas review` MVP

- Adds bounded local prepare mode for one explicitly supplied text file or stdin.
- Adds deterministic specialist recommendations without an implicit model chain.
- Adds explicit, opt-in Claude execution with an outbound-data confirmation boundary.
- Normalizes provider output through `ReviewResult` schema version 1.
- Keeps findings advisory: human review remains required.

## Safety limits

The command does not scan repositories, invoke Git, accept URLs or archives, or automatically approve, merge, release, or remediate changes. Provider execution transmits only the named input after explicit confirmation. Raw provider responses remain diagnostic-only and are not stable output.
